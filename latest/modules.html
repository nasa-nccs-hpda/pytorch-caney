pytorch_caney.processing.get_rand_patches_aug_augcond(img, mask, n_patches=16000, sz=256, nclasses=6, over=50, nodata_ascloud=True, nodata=-9999, method='augcond')[source]

Generate training data. :param images: ndarray in the format (w,h,c). :type mask: :param mask: integer ndarray with shape (x_sz, y_sz) :type n_patches: :param n_patches: number of patches :type sz: :param sz: tile size, will be used for both height and width :type nclasses: :param nclasses: number of classes present in the output data :type over: :param over: number of pixels to overlap between images :type nodata_ascloud: :param nodata_ascloud: convert no-data values to cloud labels :type method: :param method: choose between rand, cond, cloud

aug - select N * 8 number of random patches for each

image after data augmentation.

augcond - select N * 8 number of random patches for

each image, with the condition of having 1+ per tile, after data augmentation.

Return type:

array

Returns:

two numpy array with data and labels.

rand - select N number of random patches for each image cond - select N number of random patches for each image,
